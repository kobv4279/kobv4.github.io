마이나스 PC 서버

내 사진을 내 PC로 백업하려면, PC에 이 서버를 실행하세요.

① 폰에 앱 설치 (안드로이드)

마이나스 앱 다운로드 APK · 안드로이드 · "출처를 알 수 없는 앱" 경고 뜨면 허용

② PC에 서버 설치 (Windows)

Windows 서버 다운로드 ZIP · 약 54MB · Windows 10/11 · .NET 설치 불필요

폰과 PC가 같은 Wi-Fi에 있어야 연결됩니다.

PC 서버 설치 방법 (3단계)

1
압축 풀고 실행내려받은 ZIP을 풀고 WifiBackupServer.exe 를 더블클릭하세요.
2
방화벽 허용처음 실행 시 방화벽 팝업이 뜨면 액세스 허용을 누르세요. 같은 Wi-Fi의 폰이 서버를 찾으려면 필요합니다.
3
폰 앱에서 연결마이나스 앱 → 설정 → 서버 자동 검색 → 관리 페이지(http://localhost:5099)에 표시된 페어링 코드 6자리 입력. 끝!

⚠️ "Windows가 PC를 보호했습니다" 창이 뜨나요?

놀라지 마세요 — 정상입니다. 바이러스가 아니라, 아직 마이크로소프트에 등록되지 않은 새 프로그램이라 나오는 경고입니다.

A
"추가 정보" 를 클릭파란 창 가운데 있는 추가 정보 글씨를 누르세요.
B
"실행" 버튼 클릭추가 정보를 누르면 아래에 실행 버튼이 새로 나타납니다. 그걸 누르면 됩니다.

※ ZIP 안의 먼저읽어주세요.txt 에도 같은 안내가 있습니다.

💡 PC가 켜져 있고 같은 Wi-Fi에 있을 때 백업됩니다. 밤에 PC를 켜두면 그동안 자동으로 백업돼요. 백업 저장 위치는 관리 페이지(http://localhost:5099)에서 바꿀 수 있습니다.
